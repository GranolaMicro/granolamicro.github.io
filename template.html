Images must have a width tag of 128px and a border tag of 0, as well as an alt tag describing the image. If transparent, they must also have a style tag that shows a background color (default white) Additionally they can have a title tag. This link has a title tag. Use title tags to describe things in further detail, like usernames for a social media link.

If a tag is outside the master div, it goes to the edge and that looks yucky >P

Especially when it's a link!

This will allow us to style it using IDs and classes in the stylesheet, or the style tag in the div itself.
